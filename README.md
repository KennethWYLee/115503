# 智慧顧問協作洞察工作台

GitHub repository name: `115503`

第 115503 組系統已整理為可操作的靜態前端工作台，包含完整權限的核心作業、狀態管理與本機瀏覽器資料保存。

## 系統能力

- 匯入問卷、訪談與培訓回饋後，自動產生分類、關鍵字與風險摘要。
- 管理 Prompt 模板，讓不同顧問專案能沿用一致的分析標準。
- 將非結構文字轉為可搜尋、可追蹤、可比較的專案資料。
- 提供資料上傳、結果檢視、摘要輸出與分析紀錄管理。

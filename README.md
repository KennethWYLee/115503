# 智慧顧問協作洞察工作台

GitHub repository name: `115503`

此公開展示版依據第 115503 組學生文件的系統目的、功能與預期成果重新設計，重點是讓版型、視覺語言與操作情境更貼近該系統的產品定位。

## 展示重點

- 建立 AI 分析助理，結合 TF-IDF 關鍵字與 Gemini API 產出分類回饋。
- 建立 Prompt 模板庫，讓培訓回饋分析可重複、可追蹤、可調校。
- 將問卷與文字回饋轉為具分類標籤與分析紀錄的結構化資料。
- 提供專案管理介面，支援資料上傳、結果檢視與檔案輸出。
